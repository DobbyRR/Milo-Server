# Milo-Server
## Virtual Factory OPC UA Server (MES 시뮬레이션)
## 개요
본 레포지토리는 Eclipse Milo 라이브러리를 활용하여 구현한 가상 공장(Virtual Factory)용 OPC UA 서버입니다.  
본 서버는"CtroLine(MES + ERP 통합 프로젝트) 의 일부로, 실제 공장 설비 없이도 MES 로직과 데이터 흐름을 검증하기 위한  
시뮬레이션 목적으로 개발되었습니다.
## 개발 목적
- 실제 설비 없이 MES 개발 및 테스트 환경 구성
- OPC UA 기반 공장 데이터 구조 시뮬레이션
- MES - 설비 간 통신 및 데이터 흐름 사전 검증
## 주요 기능
- Eclipse Milo 기반 OPC UA Server 구현
- 가상 공장 설비 노드 및 데이터 모델링
- 노드 값 변경을 통한 생산 데이터 시뮬레이션
- MES 연동 테스트를 위한 테스트 서버 역할 수행
## 기술 스택
- Java
- Eclipse Milo (OPC UA)
- OPC UA Information Model
## 활용 시나리오
본 서버는 MES 개발 단계에서 테스트 및 검증용 시뮬레이터로 활용되었으며,  
실제 공장 설비 연동 전 단계에서 통신 구조 및 데이터 처리를 확인하는 용도로 사용되었습니다.
## 참고 사항
- 본 프로젝트느 운영 환경용 서버가 아닌 테스트/시뮬레이션 목적으로 개발되었습니다.
- 성능 최적화보다는 통신 구조 검증 및 기능 확인에 초점을 두었습니다.
